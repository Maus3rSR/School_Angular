# Support pedagogique WishFlix

Ce dossier contient le support des 5 seances (3h30 chacune) pour construire le mini Netflix "WishFlix" pas a pas.

## Progression globale

1. `seance-1.md` - Explorer le projet Angular et lier les premieres donnees.
2. `seance-2.md` - Rendre la home dynamique avec conditions, boucles et clics.
3. `seance-3.md` - Decouper l'interface en composants reutilisables.
4. `seance-4.md` - Introduire services, injection de dependances et routing.
5. `seance-5.md` - Ajouter un formulaire reactif et consolider le projet.

## Comment utiliser ce support

- Cote enseignant: suivre les demos dans l'ordre.
- Cote etudiant: realiser les exercices "version 2" apres chaque demo.
- Cote revision: utiliser les sections d'auto-evaluation en fin de seance.

## Important

Le code de depart contient des TODO relies a ces fichiers pour guider la progression sans donner une solution complete.
