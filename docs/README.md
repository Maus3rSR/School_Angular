# Support pédagogique WishFlix

Ce dossier contient le support des 5 séances (3h30 chacune) pour construire WishFlix pas à pas.

## État initial du projet

- Un seul composant: `App`.
- Un seul template monolithique: `src/app/app.template.html`.
- Données en dur dans le HTML.
- Images placeholder: `https://via.assets.so/game.png?id={id}&q=95&w=300&h=450&fit=cover`.
- Aucun pré-découpage initial en composants/pages/services/guards.

## Structure des supports

- `seance-1.md` à `seance-5.md`: cadre pédagogique par séance.
- Démos formateur: dossier `docs/demo/`.
- Exercices étudiants: dossier `docs/exercices/`.

## Progression globale

1. Séance 1: Signals + bindings + control flow depuis le HTML statique.
2. Séance 2: premier découpage en composants + `input()`/`output()`.
3. Séance 3: services + HTTP + environnements.
4. Séance 4: pages + routing lazy + guards.
5. Séance 5: formulaires réactifs + auth + interceptor.
